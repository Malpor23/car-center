# Car Center - "Frontend"

Este proyecto fue generado con  [Angular CLI](https://github.com/angular/angular-cli) en la versión 11.0.6.


## Iniciar proyecto

Ejecutar 
1. `npm install` 
2. `ng serve -o` 

### Iniciar sesión

```
Administrador: 
usuario: user-admin
contraseña: admin123

Cliente
usuario: user-client
contraseña: client123

Mecánico
usuario: user-mecanic
contraseña: mecanic123

```
